# 通用适配指南

## 使用场景
当前无法确定具体的 AI coding 工具时，使用此通用指南。

## 基本工作方式
1. 读取 README.md 了解架构结构
2. 读取 `agents/` 下的角色定义确定当前角色
3. 按照 `instructions/` 下的规则执行
4. 通过项目记忆文件协调工作

## 文件位置约定
- Agent 定义：`agents/wzx-*.md`
- 行为规则：`instructions/wzx-*.md`
- 模板：`templates/`
- 项目记忆：`.memory/`（在实际项目中）或 `projects/`（在架构仓库中）

## 核心工作流
1. 阅读角色定义，理解当前职责和边界
2. 阅读项目记忆文件（简报、状态、上下文、交接）
3. 执行分配的任务
4. 更新当前上下文和交接文件
5. 在交接中指定下一个角色和动作
